HTML

  • The head element contains information about the webpage.
  • The title element specifies a title for the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 element defines a large heading.
  • The p element defines a paragragh.

Git

  • git status:checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: commits all files to the working branch.
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot.
  • git pull origin main: pulls the base branch into feature branch ensuring it is up to date.
  • git push origin [branch name]: pushes up all of your changes.
