HTML

  • The html element is the root element of an HTML page.
  • The head element contains information about the webpage.
  • The title element specifies a title for the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 element defines a large heading.
  • The p element defines a paragragh.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A preceding period[.], designates a class selector.
  • Assigning colors can either be done semantically using color names or by using it hexidecimal code.

Git

  • git status:checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: commits all files to the working branch.
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot.
  • git pull origin main: pulls the base branch into feature branch ensuring it is up to date.
  • git push origin [branch name]: pushes up all of your changes.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • When a function is created, you are defining a block of code that performs a specific task and giving it a unique name.
  • To execute a function, you must first call it in your code.
  • For and If statements allow functions to become more durable and reusable in the project code.
